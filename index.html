C# Foundations and OO Progression

Two tracks, one path. Build understanding on the theory and micro-exercise track, then prove it on the programming track where your C# compiles and runs in the browser.

Track 1

Theory and micro-exercises

Read, predict, and write short fragments to lock in the concepts.

Recommended first

Level 1: Foundations

Why software exists, state vs behavior, variables/functions, class vs instance, encapsulation, polymorphism basics, and dependency injection intro.

Theory Micro-exercises

Bridge to SOLID

Reading Objects

A gentle step up from single-token drills. Read how a few objects work together, give one job to one place, and trace the output before you run it - the habits the SOLID drills will lean on, with no new theory.

After Level 1

Level 2: Core C# Coding Drills

Quick exercises that force writing short C# fragments: variables, functions, class/instance, encapsulation, and polymorphism basics.

Reuse - read to understand

Level 4: Reuse Without Regret

Inheritance, composition, and the polymorphism payoff - taught by reading code, not writing it. Decide is-a vs has-a, see why C# forbids two parents (the diamond problem), and learn when to favour composition. Includes a guided code tour and runnable examples.

Programming tasks

Bridge to the capstone

First Builds

Five small programs you write from scratch and run: a class, a method, a constructor, an interface, a second implementation. Each is one win, and together they are the exact moves the capstone asks for. Write the code, run it, match the output.
